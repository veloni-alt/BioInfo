### DESCRIÇÃO
Protótipo simples, porém totalmente funcional, que atua como uma interface para dados genéticos públicos, utilizando tecnologias como:
- Backend: Python (Linguagem principal para o desenvolvimento.)
- Web Framework: Flask (Estrutura leve para construir a aplicação web.)
- Consumo de API: Ensembl REST API (Biblioteca pública de dados genéticos.)
- Interface: HTML (Estrutura e estilização básica para a exibição dos dados.)
